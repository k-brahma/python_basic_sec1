## part1

1. リテラル値
    - 数値型
    - bool型
    - 文字列型
    - None型

2. 代表的な演算子(その1)  
   算術演算子
    - x + y  : 足し算
    - x - y  : 引き算
    - x * y  : 掛け算
    - x / y  : 割り算
    - x // y : 割り算して、商だけ求める
    - x % y  : 割り算して、あまりだけ求める
    - x ** y : x の y 条

3. 代表的な演算子(その2)  
   累積演算子
    - +=  : 加算
    - -=  : 減算

ちょっと高度な話:

1. Pythonでは、登場するほぼすべてが「オブジェクト」です。

    - 数値もオブジェクト
    - 文字列もオブジェクト
    - 関数もオブジェクト
    - クラスもオブジェクト
    - モジュールもオブジェクト

2. 「オブジェクト」であるとは...

    - プロパティがある
    - メソッドがある
    - 変数を使って参照できる
